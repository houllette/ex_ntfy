# ExNtfy

> 🚧 **Under construction** — this library is being built and has no usable API yet.

An Elixir SDK for [ntfy](https://ntfy.sh), the simple HTTP-based pub-sub notification
service. The goal: a small, well-tested, Req-based client for publishing notifications
and subscribing to topics on ntfy.sh or self-hosted ntfy servers.

Check back soon — documentation and usage examples will land alongside the first release.
